# SCH的简易聊天室
这是一个基于局域网通信的简易网页聊天室，使用Vue+varlet+Express开发完成，功能将持续优化。

## 下载程序
```shell
git clone https://gitee.com/schlibra/chat-vue.git
cd chat-vue
```
## 安装依赖
```shell
# npm
npm install

# yarn
yarn install
```
## 编译Vue页面
```shell
# npm
npm run build

# yarn
yarn build 
```
## 启动程序
```shell
node app.js
```
## 访问
在浏览器中访问地址：http://localhost/ ，或访问你的局域网IP
